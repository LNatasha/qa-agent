API platform with a GUI for designing, testing, and documenting REST and GraphQL APIs. Collections can be run in CI via Newman (CLI runner).

## Strengths
- Fastest tool for exploratory API testing and quick validation
- Environment variables for multi-environment testing
- Contract testing via Postman schemas and collection tests
- Newman CLI integrates with any CI pipeline
