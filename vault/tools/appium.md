Open-source mobile automation framework that drives iOS and Android apps using the WebDriver protocol. Supports native, hybrid, and mobile web apps.

## Strengths
- Single API for iOS and Android
- No app recompilation needed
- Supports real devices and simulators/emulators
- Integrates with cloud device farms (BrowserStack, Sauce Labs)

## When to prefer over Detox
Use Appium for cross-platform coverage or when the team already uses WebDriver. Use [[detox]] for React Native apps where grey-box testing gives faster, more reliable results.
