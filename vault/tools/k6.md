Grafana k6 is a developer-centric load testing tool with a JavaScript/TypeScript scripting API. Runs locally or in k6 Cloud.

## Strengths
- TypeScript-native scripting — test code lives in the same repo as app code
- Rich metrics: VUs, RPS, P95/P99 latency, error rates
- Threshold assertions fail CI pipelines automatically when SLAs are breached
- Supports load, stress, spike, and soak scenarios in one script
