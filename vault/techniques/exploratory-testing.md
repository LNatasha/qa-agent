Simultaneous learning, test design, and execution — the tester uses domain knowledge and intuition to probe the system without a predefined script.

## When to use
New features without stable specs, post-regression safety net, UX flows, and areas where automation coverage is thin.

## Strengths
- Finds defects that scripted tests miss
- Fast to start — no upfront test design
- Excellent for mobile apps with complex gesture interactions
